## 摘要
尽管基于可验证奖励的强化学习（RLVR）提升了大语言视觉模型（LVLMs）的推理能力，但主流框架存在一个基础性的方法论缺陷：通过在所有生成的 Token 上分配相同的优势值（advantages），这些方法本质上稀释了优化多模态推理中关键的、基于视觉的步骤所需的学习信号。为了弥补这一差距，我们定义了“Token 视觉依赖度”（Token Visual Dependency），通过计算视觉条件预测分布与纯文本预测分布之间的 Kullback-Leibler (KL) 散度，来量化视觉输入的因果信息增益。研究发现这种依赖关系具有高度稀疏性且在语义上至关重要，据此我们提出了感知驱动策略优化（PGPO），这是一种新型的细粒度信用分配框架，能够在 Token 级别动态重塑优势值。通过一种阈值门控、质量守恒的机制，PGPO 主动放大视觉依赖型 Token 的学习信号，同时抑制来自语言先验的梯度噪声。基于 Qwen2.5-VL 系列在七个具有挑战性的多模态推理基准测试上的广泛实验表明，PGPO 平均将模型性能提升了 18.7%。理论和实证分析均证实，PGPO 有效降低了梯度方差，防止了训练崩溃，并作为一种强有力的正则化工具，促进了稳健的、基于感知的多模态推理。代码将发布于 https://github.com/Yzk1114/PGPO。

## Abstract
While Reinforcement Learning from Verifiable Rewards (RLVR) has advanced reasoning in Large Vision-Language Models (LVLMs), prevailing frameworks suffer from a foundational methodological flaw: by distributing identical advantages across all generated tokens, these methods inherently dilute the learning signals essential for optimizing the critical, visually-grounded steps of multimodal reasoning. To bridge this gap, we formulate \textit{Token Visual Dependency}, quantifying the causal information gain of visual inputs via the Kullback-Leibler (KL) divergence between visual-conditioned and text-only predictive distributions. Revealing that this dependency is highly sparse and semantically pivotal, we introduce Perception-Grounded Policy Optimization (PGPO), which is a novel fine-grained credit assignment framework that dynamically reshapes advantages at the token level. Through a threshold-gated, mass-conserving mechanism, PGPO actively amplifies learning signals for visually-dependent tokens while suppressing gradient noise from linguistic priors. Extensive experiments based on the Qwen2.5-VL series across seven challenging multimodal reasoning benchmarks demonstrate that PGPO boosts models by 18.7% on average. Both theoretical and empirical analyses confirm that PGPO effectively reduces gradient variance, prevents training collapse, and acts as a potent regularizer for robust, perception-grounded multimodal reasoning. Code will be published on https://github.com/Yzk1114/PGPO.

#### 1. 核心问题与整体含义（研究动机和背景）
在大型视觉语言模型（LVLMs）的强化学习（RLVR）过程中，现有的框架（如 GRPO）通常采用**粗粒度的信用分配机制**：将整个序列的奖励（Reward）平均分配给生成过程中的每一个 Token。
然而，作者发现多模态推理中并非所有 Token 都同等重要。例如在解决几何题时，提取图形数字的 Token 极度依赖视觉，而连接词或语法 Token 则更多依赖语言先验。这种“吃大锅饭”的分配方式稀释了关键视觉感知步骤的学习信号，导致模型容易产生幻觉或过度依赖文本偏见。本文的研究动机在于：**如何量化并强化那些真正依赖视觉感知的关键推理步骤？**

#### 2. 方法论：感知驱动策略优化（PGPO）
核心思想是通过量化每个 Token 对视觉信息的“依赖程度”，动态调整其在强化学习中的权重（Advantage）。

*   **Token 视觉依赖度量化**：利用 KL 散度计算“有视觉输入”和“无视觉输入”（通过 Attention Mask 屏蔽视觉 Token）两种情况下模型预测分布的差异。KL 散度越大，说明该 Token 越依赖图像信息。
*   **两阶段信号处理**：
    1.  **对数压缩与归一化**：对原始 KL 散度进行对数处理以平滑长尾分布，并进行序列内的 Min-Max 归一化，得到有界的依赖得分 $I_t$。
    2.  **阈值门控优势重塑（Threshold-Gated Advantage Reshaping）**：设置阈值 $\tau$，将 Token 分为“视觉依赖型”和“噪声型”。对于依赖型 Token 放大其学习信号（Boost），对于噪声型 Token 抑制其梯度（Suppress）。
*   **质量守恒归一化（Mass-Conserving Normalization）**：为了保证训练稳定性，PGPO 确保重塑后的 Token 权重总和与原始序列长度一致。这在数学上保证了优势值的均值保持为零，防止了梯度爆炸或训练崩溃。

*   **模型基础**：采用 Qwen2.5-VL-3B 和 7B 作为基座模型。

*   **对比方法**：包括基础模型、标准 GRPO、DAPO、以及最新的多模态 RL 方法如 PAPO、VPPO、MM-Eureka 等。

*   **超参数分析**：对阈值 $\tau$ 和增强系数 $\beta$ 进行了敏感性测试。
*   **客观性**：所有评估均采用**精确匹配（Exact Match）**而非 LLM-as-a-judge，确保了结果的客观性和可重复性。

#### 6. 主要结论与发现
*   **性能飞跃**：PGPO 在 7 个基准测试中均达到 SOTA，相比基座模型平均提升 **18.7%**，显著优于 VPPO 和 PAPO。
*   **训练稳定性**：PGPO 有效解决了 DAPO 等方法在训练后期容易出现的“熵爆炸”和性能崩溃问题。
*   **感知增强**：实证分析显示，随着训练进行，模型生成的 Token 视觉依赖度稳步上升，说明模型确实学会了“多看图、少瞎猜”。

#### 7. 优点

*   **即插即用**：该框架可以轻松集成到现有的 GRPO 或 PPO 流程中，具有较强的通用性。

*   **超参数敏感性**：虽然 $\tau$ 和 $\beta$ 在实验中表现稳健，但在不同规模或架构的模型上可能需要重新微调。

*   **模型规模限制**：目前实验最高仅验证到 7B 参数规模，在超大规模模型上的表现尚待观察。
