## 摘要
对于长程LLM智能体而言，记忆是一份关于每个未来决策被允许看到的内容的契约。最简单的契约将过去的观察、工具调用和反思附加到每个提示中，这使得先前的上下文易于访问，但也使其变成一团混乱的混合物，难以孤立任何单个记忆成分的影响。我们引入并装备了一种替代的有界契约：每个决策都由通过类型化检索组装的新用户消息做出，不附加任何原始的跨决策记录。因此，在任何长度的运行中，提示都保持有界，并且任何单个层都可以被单独消融。我们在《杀戮尖塔2》中实例化了该契约，这是一款封闭规则、随机的卡组构建游戏，其运行需要数百个战术和战略决策。在同一游戏上，针对前沿LLM的公开在线基准测试报告，在最低难度下，五种配置的胜率为零，而开发者报告的相同难度下的人类胜率为16%；这项任务很困难但尚未饱和。在我们的框架内，固定A0消融显示，当启用触发式战略技能时，观察到的差异最大：无存储基线赢得3/10局游戏，而添加技能层后赢得6/10局。在此样本量下，该比较是方向性的而非统计上决定性的（费舍尔精确检验p≈0.37）；跨骨干探查和公开的累积上下文基线被报告为操作比较，而非对契约变量本身的控制测试。我们发布了一个可复现的测试平台：298条带有条件标签的完整轨迹、冻结的记忆/技能快照、提示记录和分析脚本——一个智能体设计以及一种经过验证、可复用的方法，用于研究显式记忆层如何塑造长程LLM智能体的决策。

## Abstract
Memory for a long-horizon LLM agent is a contract about what each future decision is allowed to see. The simplest contract appends past observations, tool calls, and reflections to every prompt, which makes prior context easy to access but also turns it into a jumbled mixture in which the effect of any single memory component is hard to isolate. We introduce and instrument an alternative bounded contract: every decision is made from a fresh user message assembled by typed retrieval, with no raw cross-decision transcript appended. The prompt thus stays bounded across runs of any length, and any single layer can be ablated in isolation. We instantiate the contract in Slay the Spire 2, a closed-rule stochastic deck-building game whose runs require hundreds of tactical and strategic decisions. A public online benchmark of frontier LLMs on the same game reports zero wins at the lowest difficulty across five configurations, and the developer-reported human win rate at the same difficulty is 16%; the task is hard but not saturated. Within our harness, a fixed-A0 ablation shows the largest observed difference when triggered strategic skills are enabled: the no-store baseline wins 3/10 games and adding the skill layer 6/10. At this sample size the comparison is directional rather than statistically decisive (Fisher exact p\approx0.37); a cross-backbone probe and public accumulating-context baselines are reported as operational comparisons rather than controlled tests of the contract variable itself. We release a reproducible testbed: 298 completed trajectories with condition tags, frozen memory/skill snapshots, prompt records, and analysis scripts -- an agent design and a validated, reusable methodology for studying how explicit memory layers shape long-horizon LLM-agent decisions.

### 1. 论文的核心问题与整体含义

*   **研究动机与背景**：
    *   当前长程LLM智能体的记忆机制普遍采用**上下文累积**模式，即不断将历史观察、工具调用和反思追加到提示中。
    *   这种方式虽然能让模型看到历史，但存在**两大弊端**：
        1.  **上下文无限膨胀**：运行步数变长后，提示可能会超出模型上下文窗口限制，并导致计算成本急剧增加。
        2.  **归因困难**：所有历史信息混杂在一起，无法单独评估某个记忆组件（如技能、经历、规则）对最终决策的贡献，导致记忆成为一个“黑箱”。
    *   因此，核心问题在于：**是否可以将记忆接口设计为有界、可检查且可消融的，从而将记忆本身变成一个受控的研究对象？**

### 2. 论文提出的方法论

*   **核心思想**：
    *   提出一种**有界记忆契约**：智能体在每个决策点收到的提示都是**从零重新组装**的，绝不附加原始的跨决策历史记录。
    *   这种设计使提示长度恒定，不受运行长度影响，并且任何记忆层都可以被单独关闭（消融），以测试其效果。

*   **关键技术细节**：
    *   **类型化检索与五层记忆架构**：每个决策的用户消息由五个类型化知识槽位的内容组装而成：
        *   **L1 (Protocol)**：不可变的角色和协议模板。
        *   **L2 (Schema)**：不可变的状态特定架构和合法行动格式。
        *   **L3 (Rules)**：可检索的枚举型游戏规则和实体数据。
        *   **L4 (Episodes)**：可写的**情景记忆**，存储运行后的总结。
        *   **L5 (Skills)**：可写的**技能库**，存储基于触发条件的战略性行为指南。
    *   **消融实验设计**：通过开关L4和L5，可以清晰地测试不同记忆层的影响。这是该方法的关键实验优势。
    *   **决策路由**：并非所有操作都调用LLM。一个路由分发器将简单的、战术性的战斗动作交给快速层处理，只有需要战略决策时才调用主LLM，从而大幅减少了LLM调用次数。
    *   **技能发现与更新**：L5技能通过两种方式填充，(a) 基于失败的蒸馏式自我进化；(b) 基于模板的填充（模式B），以此验证技能层的有效性独立于技能的来源。

### 3. 实验设计

*   **数据集/测试场景**：
    *   **《杀戮尖塔2》**：一个封闭、随机、回合制的卡组构建游戏。
    *   **选择该场景的原因**：
        *   **长程**：一局游戏需数百个连续决策，适合测试长程记忆。
        *   **文本可读**：游戏状态、规则、卡牌等均为文本，可被LLM直接理解，无需视觉输入。
        *   **高难度但未饱和**：在最低难度（A0）下，公开基准中前沿模型胜率为零，而人类胜率为16%，说明任务极具挑战性且有提升空间。
*   **Benchmark与对比基线**：
    *   **主要比较对象**：在相同测试平台下，进行**内部消融**，比较不同记忆层配置的性能。
        *   `baseline-strict`：无任何存储（L4/L5）的严格基线。
        *   `prompt-only`：具有完整提示但禁用L4和L5。
        *   `mode-a`：启用人工编写的L5技能种子库。
        *   `mode-b-frozen`：启用模板填充的L5技能。
        *   `full-frozen`：启用L4和L5（冻结模式）。
    *   **外部校准基线**：
        *   **AGI-Eval**：公开的LLM基准，报告其在同款游戏A0难度下零胜率，用于校准任务难度。
        *   **开发者数据**：Mega Crit报告的16%社区玩家胜率，同样用于难度校准。
        *   **公开累积上下文智能体**：STS2MCP和CharTyr，这两个系统遵循传统的累积上下文设计。此比较是操作性的，旨在展示当前实践现状，而非严格的变量控制。

    *   **公平性**：在与外部累积上下文智能体比较时，作者强调了这是一个“操作性”而非“受控”比较，因为对比双方在游戏补丁、路由、推理参数等方面存在差异，无法将性能差异完全归因于记忆契约本身。这种坦诚的声明保证了比较的公平性。

### 6. 论文的主要结论与发现

*   **L5技能层是关键**：在固定A0难度的消融实验中，最大的观察差异与启用手动（或模板填充的）L5战略技能层相关，将无存储基线的3/10胜率提升至6/10。
*   **方向性而非决定性**：上述胜率差异在统计学上不显著，被定性为方向性结果。这说明有界契约下的技能层表现出积极信号，但需要更大规模实验确认。
*   **L4情景记忆效果饱和**：在A0难度下，增加L4情景记忆存储未能进一步提升胜率，表明在当前设定下L5的增益已接近瓶颈。
*   **技能具有模型敏感性**：基于Gemini生成的L5技能库在Qwen模型上带来了分数提升，但在DeepSeek模型上导致分数下降，说明这类技能知识无法轻易地跨模型转移。
*   **有界契约的工程优势**：与公开的累积上下文智能体相比，AgenticSTS不仅在胜率上大幅领先，且在速度和成本（Token消耗）上展现出一到两个数量级的优势。

### 7. 优点

*   **方法学创新**：将有界、类型化、可消融的记忆契约概念化并实现，将LLM智能体的记忆从一个实现细节提升为可系统研究的对象。
*   **实验设计的严谨性**：设计了多层次的实验流（固定难度消融、跨模型探针、难度阶梯），并明确区分了内部受控消融与外部操作性比较，对结论的适用范围界定清晰。
*   **高可复现性**：详细描述了测试平台、记忆快照、轨迹数据和统计分析脚本都公开发布，为社区提供了坚实的研究基础。
*   **任务选择的优势**：选择《杀戮尖塔2》作为测试平台极具巧思，既提供了长程、随机的决策挑战，又因其文本化的状态和规则而完美适配LLM评估，避免了视觉处理等额外复杂性。

### 8. 不足与局限

*   **样本量有限**：核心消融实验的样本量（n=10/格）不足，导致结论仅为方向性的，无法得出统计上确定的因果推断。
*   **缺乏关键对照实验**：最受控的对比应是在完全相同的代码库、骨干网络和参数下，比较有界契约与累积上下文契约的差异，但本工作未包含此实验。这被视为一项未来工作。
*   **结果跨模型泛化能力存疑**：跨骨干网络探针表明，高度优化的L5技能库效果依赖于特定模型，泛化能力有限。
*   **测试覆盖范围窄**：
    *   **单角色**：主要在一个游戏角色（The Silent）上进行测试。
    *   **单任务类型**：结论局限于回合制、封闭规则、文本状态的游戏环境，向开放域、连续控制或视觉任务迁移的有效性未知。
*   **技能库的自主性受限**：L5技能在很大程度上仍依赖人工编写的种子和模板，并非完全从零自主学习，其自主演化能力的上限有待检验。
