## 摘要
我们提出了 FootsiesGym，一个用于在非平凡双人零和非完美信息博弈中进行学习的开源环境。它基于 HiFight 的极简 2D 格斗游戏 Footsies，分离出格斗游戏立回中循环、非传递的策略交互，同时保持足够的简洁以进行高效分析。我们提供了一个向量化模拟器，能在标准硬件上实现高通量训练，使环境易于访问且可复现。我们描述了环境的设计，对若干强化学习算法进行了基准测试，并讨论了它所支持的前沿研究方向。代码可在 https://github.com/como-research/FootsiesGym 获取。

## Abstract
We present FootsiesGym, an open-source environment for learning in a non-trivial two-player, zero-sum, imperfect-information game. Built on HiFight's minimalist 2D fighting game Footsies, it isolates the cyclic, non-transitive strategic interactions of fighting game neutral play while remaining simple enough for efficient analysis. We provide a vectorized simulator that enables high-throughput training on standard hardware, making the environment accessible and reproducible. We describe the design of the environment, benchmark several reinforcement learning algorithms, and discuss open research directions it enables. The code is available at https://github.com/como-research/FootsiesGym.

### 2. 论文提出的方法论

*   **核心思想**
    *   以极简主义设计隔离格斗游戏的**非传递性战略结构**。通过移除复杂的连招机制，游戏完全专注于“立回”——即双方玩家通过移动、攻击和预判来创造进攻机会的阶段。
*   **环境设计细节**
    *   **游戏机制**：角色无血量条，被特殊攻击击中一次即判负；普通攻击可被防御三次；包含冲刺、普通攻击和需蓄力的特殊攻击。
    *   **动作空间**：离散动作，默认为6个（如空操作、移动、攻击），可选扩展至9个（增加蓄力动作）。
    *   **观察空间**：一个85维（可扩至88维）的特征向量，包含双方的位置、速度、状态等信息。为模拟信息不对称（人类玩家无法看到对手手柄输入），观测中包含了**特权自特征**（如冲刺可用性、蓄力进度、上一步动作），对手无法获取。
    *   **奖励函数**：零和奖励，胜/负获得±w（默认w=10）的终止奖励，超时平局奖励为0。可选密集奖励。
    *   **动作延迟**：一个关键参数，指从动作选择到游戏中执行之间的帧数。通过调整延迟，可以控制攻防的“可反应性”与“须预判性”，从而塑造策略深度。零延迟导致退化策略，高延迟导致近似随机。
*   **技术实现与向量化**
    *   游戏逻辑基于C#的Unity引擎，但训练时**解耦为无渲染的头less模式**。
    *   **多层并行架构**：
        1.  **进程内并行**：单个C#进程同时管理K个独立游戏实例。
        2.  **多进程并行**：Python客户端通过gRPC与多个独立游戏服务进程通信。论文展示了在24核/48线程工作站上使用4个进程、每进程128个实例时，吞吐量可达**约52,500环境步/秒**。
    *   遵循PettingZoo API标准，易于与现有强化学习框架集成。

### 3. 实验设计

*   **使用的场景/数据集**
    *   所有实验均在 **FootsiesGym 环境**本身中进行，这是一个合成基准，不涉及外部数据集。
*   **基准测试的算法**
    *   **PPO (Sched.)**：带熵系数退火调度的近端策略优化。
    *   **PPO**：固定熵系数的PPO。
    *   **EMAgnet**：一种基于参数空间指数移动平均正则化的策略梯度方法，应用于PPO。
    *   **PFSP**：基于优先虚拟自我对弈（从自身历史检查点中采样对手）。
*   **对比与评估方式**
    *   **对抗启发式对手**：评估策略对战“均匀随机”和“无操作”对手的胜率，衡量基本技能。
    *   **近似的可利用性**：通过训练**近似最优响应**来对抗已冻结的策略，以其成功率衡量策略的鲁棒性。这是复杂博弈中计算真实可利用性的常用近似方法。
    *   **头对头对战**：通过不同算法策略间的互相对战，评估相对技能水平和策略特性。
    *   **技能发现**：通过统计“方向特殊攻击”指令的执行频率，评估算法在稀疏奖励下发现和执行复杂技能的能力。

### 6. 论文的主要结论与发现

*   **环境成功复现博弈特征**：FootsiesGym 成功捕捉了格斗游戏的循环、非传递策略交互，提供了一个有效的中间复杂度测试平台。
*   **性能与技能的矛盾**：
    *   所有算法的策略在变得更强的同时，都表现出**更强的反应性和更低的主动性**，例如面对不动对手时胜率下降（平局增多）。
    *   **EMAgnet** 虽然在对战PPO时取得了正的回报，但在近似可利用性上表现最差，揭示了技能与鲁棒性（不易被利用）之间的潜在冲突。**PFSP** 相对表现更均衡。
*   **复杂技能难以发现**：在标准配置下，所有基线算法都**几乎无法稳定地发现并持续使用需要长时间蓄力的“方向特殊攻击”**。这表明在高熵或强正则化的训练范式下，有价值但难以探索的策略可能被排除出策略空间。
*   **动作延迟的关键影响**：`action_delay` 参数能有效调控策略的风格，从脆弱的纯反应型（低延迟）到依赖预判型（高延迟），是塑造游戏深度的关键杠杆。

### 7. 优点

*   **填补领域空白**：精准定位于过于简单和过于复杂的环境之间，为研究非传递性博弈提供了一个在复杂度和算力上均适宜的“甜点级”基准。
*   **设计精巧**：通过极简主义设计，成功隔离了格斗游戏立回的核心博弈结构，使研究可以聚焦。
*   **高性能与可复现性**：开源且提供高效的向量化模拟器，支持在标准单机上实现高通量训练，极大地降低了研究门槛，保证了实验的可复现性。
*   **暴露关键科学问题**：实验清晰揭示了强化学习训练中“技能 vs 主动性”、“技能 vs 鲁棒性”以及“探索 vs 技能发现”等深刻矛盾，为未来研究指明了具体方向。

### 8. 不足与局限

*   **复杂度差距**：Footsies是极简游戏，与现代商业格斗游戏（拥有复杂连招系统、更丰富角色等）在复杂度上仍有巨大鸿沟，其结论向更复杂游戏的泛化性有待验证。
*   **评估限制**：利用**近似最优响应**来估算可利用性，这仅是真实可利用性的一个下界，可能无法完全揭示策略的脆弱性。
*   **基线不充分**：基线算法**未经过系统性的精细调优**，因此基于当前实验的算法间对比结论（例如EMAgnets更易被利用）是启发性的，不能作为定论。
*   **算力信息缺失**：未报告训练基线算法所使用的具体GPU资源和时间，这在一定程度上降低了其他研究者复现或对比训练成本的精确性。
