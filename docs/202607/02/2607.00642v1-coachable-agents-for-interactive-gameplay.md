## 摘要
强化学习已被证明是创建先进人工智能和机器人系统的宝贵工具，在游戏玩法、机器人技术乃至基础模型等领域均有贡献。通过试错过程，这些人工智能系统通常学习一种接近最优的行为来解决其任务。然而，在许多用例中，人们希望在一定程度上对任务的解决方式施加控制，最好能实时进行。我们将对核心任务的这些修改称为风格。我们将通用价值函数逼近器（UVFA）与精心选择的训练场景、学习算法和数据增强相结合，创建了一个用于训练智能体的框架，使其在复杂领域中展现出风格。我们展示了该框架在AAA视频游戏《地平线：西之绝境》和《GT赛车》以及一个开源人形机器人测试领域中的应用。尽管这些领域性质各异——赛车、风格化游戏战斗和人形行走——每个智能体在满足其领域主要任务的同时，对风格请求都表现出强烈的一致性。重要的是，本文概述的技术允许最终用户在运行时选择最终行为，从而灵活控制最终的执行表现。

## Abstract
Reinforcement learning has proven to be a valuable tool in the creation of advanced AI and robotic systems, contributing to everything from game playing to robotics to foundation models. Through trial-and-error, these AI systems typically learn one, near-optimal behavior to solve their tasks. However, there are many use cases in which one would like to assert some level of control, preferably in real time, over how the task is solved. We refer to these modifications of a core task as styles. We combine universal value function approximators (UVFAs) with carefully selected training scenarios, learning algorithms, and data augmentation to create a framework for coaching agents that exhibit styles in complex domains. We demonstrate the framework's application in the AAA video games Horizon Forbidden West and Gran Turismo, and in an open-source humanoid test domain. Despite the different nature of the domains -- car racing, stylized game combat, and humanoid walking -- each agent shows strong coherence to the style requests while still satisfying the main task in its domain. Importantly, the techniques outlined in this paper allow an end user to choose the final behavior at run time, giving them flexible control over the final executed performance.

### 2. 论文提出的方法论

- **核心思想: 风格条件化通用价值函数逼近器**
  - 基于**通用价值函数逼近器（UVFA）** 的概念，但对其进行了关键性改造。不同于原始的UVFA通过“目标”来区分不同任务，本文将其拓展为通过“风格向量”来区分同一任务的不同执行方式。
  - **奖励函数分解**: 奖励函数被形式化地分解为两部分：
    - **任务奖励**：固定不变，定义核心目标（如击败敌人）。
    - **风格奖励**：由多维风格向量 控制，包含线性权重、阈值和目标设定点等，用于鼓励或抑制特定行为（如使用特定武器、造成元素伤害、节省轮胎）。
  - **策略与价值函数条件化**: 将编码后的风格向量 作为额外输入，与状态 拼接形成增强状态 。策略 和价值网络 都以此增强状态为条件，从而让单一网络能够表达多种行为模式。

- **关键技术细节与算法流程**
  - **风格向量 的构成**:
    ```
    r_{style}(s, a; \theta) = \sum_{i=1}^{m} w_i g_i(s, a; \xi_i), \theta = (w_1, ..., w_m, \xi_1, ..., \xi_m)
    ```
    其中， 是标量权重， 是风格奖励函数， 是风格参数向量。权重可以取正值以鼓励、负值以抑制或零值以表示无偏好。
  - **训练策略**:
    - **情景式（Episodic）风格采样**: 在每个训练回合开始时，从预定义的分布中采样风格向量，并在该回合内保持不变。这避免了因运行时风格序列未知而产生的信用分配问题。
    - **任务模仿**: 一种**数据增强**技术。在每次策略/价值网络更新时，对从经验回放缓冲区采样的一批转移数据，除了使用其原始风格标签外，还会**额外生成多个备选风格向量**并重新计算其风格奖励，从而创造出`(n+1)`个标记版本的转移数据。这极大地提高了数据效率，让智能体从相同的交互数据中学习到更广泛的风格响应，而无需将这些伪造数据写回缓冲区。
    - **分层抽样**: 使用**多表**或**事件表**等特殊经验回放缓冲区，确保小批量数据在场景或风格配置上的平衡采样，避免训练被主导性行为所主导。
  - **针对复杂领域（HFW）的算法创新: Categorical Regularized Actor-Critic (Cat-RAC)**
    - 为处理《地平线》游戏中混合动作空间（离散按钮 + 连续摇杆/扳机），对SAC算法进行泛化，提出**RAC**目标，用正则化项替代传统的熵项。
    - 引入**分类分布价值网络（Cat-RAC）**，将值函数转化为离散分类分布并通过交叉熵进行训练，提高了对高方差、多模态目标的鲁棒性。
    - 对离散动作头，采用**Gumbel-Softmax宽松化**和**Regret Matching**梯度修正技巧，实现了从价值网络到策略的稳定梯度传播。
